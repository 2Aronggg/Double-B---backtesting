# Double-B---backtesting
5분봉 기반의 Double-B(20/2, 4/4) 지표로 단기 변동성을 포착하고, RSI 3단 콤보와 20·60·120 이평선 배열을 조합하여 추세의 에너지와 방향성을 검증한다. 캔들이 두 밴드에 동시 접촉하고 RSI 변곡 신호와 이평선 정·역배열이 일치할 때만 진입하며, 강한 추세 시 역매매를 금지하고 거래당 0.1%의 보수적 비용을 반영한다. 2023년부터 2026년까지의 데이터를 바탕으로 주식, 원자재, 가상자산 등 16개 대상의 자산군·시장·사이즈별 전략 유효성을 통계적으로 분석한다.
