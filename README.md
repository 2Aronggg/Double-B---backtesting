# Double-B: Quantitative AI Research & Analysis System

> **AI 기반 시장 차트 분석 및 다각적 백테스팅 엔진**
>
> 본 프로젝트는 '더블비(Double-B)' 지표를 핵심으로 하여 자산군별, 시장별, 사이즈별 변동성과 모멘텀을 통계적으로 분석하는 리서치 프레임워크이다.

---

## 🔬 Analysis Methodology: Bollinger Band + @
시장 차트의 통계적 특성을 분석하기 위해 다음과 같은 기술적 지표와 조건값을 설정하여 수익률의 추세 및 비추세 구간을 정의한다.

### 1. 지표 및 조건 상세
| 구분 | 지표/조건 | 설정값 및 목적 |
| :--- | :--- | :--- |
| **타임프레임** | 5분봉 | 단기 변동성 포착 및 정밀 진입 시점 결정 |
| **변동성 1** | Band A (더블비) | 20주기 / 2 표준편차 / 종가 기준 (일반적 가격 변동 범위 확인) |
| **변동성 2** | Band B (더블비) | 4주기 / 4 표준편차 / 시가 기준 (극단적 단기 변동성 폭발 포착) |
| **모멘텀** | RSI | 과매수(70) / 과매도(30) 기준을 통한 추세 에너지 측정 |
| **추세 필터** | 이동평균선 (MA) | 20, 60, 120 주기를 활용한 메인 트렌드 방향성 확인 |

---

## 📊 Analysis Targets
데이터의 범위, 시장의 성격, 전략의 유효기간을 고려하여 총 16개의 대상을 분석한다.

### 1. 자산군 별 분석 (Asset Class) - 총 7개
* **주식:** KOSPI 200, KOSDAQ 레버리지
* **원자재:** 금(Gold), 구리, 원유 (공급망 및 인플레이션 기반 추세 분석)
* **가상자산:** BTC-USD (Large-cap), SOL-USD (Mid-cap)

### 2. 사이즈 별 분석 - 총 4개
* **대형주:** S&P 500, KOSPI 시가총액 1위 ~ 100위 (KOSPI 100)
* **중소형주:** Russell 2000, KOSPI 시가총액 300위 이하 또는 KOSDAQ 전종목

### 3. 시장 별 분석 (Market) - 총 4개
* **선진국 시장 (Developed):** 미국(NASDAQ), 독일(DAX)
* **신흥국 시장 (Emerging):** 한국(KOSPI), 인도(NIFTY 50)

## 📅 Backtesting Environment
전략의 견고함을 검증하기 위해 다음과 같이 데이터 기간을 분리하여 학습 및 테스트를 진행한다.

* **훈련 데이터 (In-Sample):** 2023년 ~ 2024년 (2개년)
* **테스트 데이터 (Out-of-Sample):** 2025년 ~ 2026년 (2개년)

## 🛠 System Architecture
본 프로젝트는 웹 기반 대시보드를 통해 리서치 결과를 시각화한다.

* **Backend:** Python 기반 데이터 전처리 및 통계 연산 루틴
* **Frontend:** HTML/JS 기반 실시간 차트 렌더링 및 분석 결과 시각화
* **Analysis Engine:** 보유기간별/자산군별/사이즈별 수익률 통계 분석 모듈

## 🔍 Key Findings
* **가상자산군(BTC, SOL):** 높은 변동성을 바탕으로 가장 높은 누적 수익률과 Sharpe Ratio를 기록했으나, MDD 관리 측면에서 리스크가 확인됨.
* **신흥국 시장(NIFTY 50):** 선진국 대비 낮은 변동성과 꾸준한 우상향 추세를 보이며 안정적인 성과를 기록함.
* **더블비 지표 유효성:** Band B(4주기/4표준편차) 돌파 시 단기 모멘텀이 발생하는 구간에서 승률이 유의미하게 상승함이 확인됨.
